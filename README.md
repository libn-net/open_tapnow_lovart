## 功能介绍

#### 1.0.1231-pre版
- ✅ 1、工作流：支持批量生成营销图片、营销视频、AI漫剧、AI短剧 
- ✅ 2、图片卡片：支持Nano Banana/MJ/等多个优秀模型 
- ✅ 3、视频卡片：支持Sora2，Veo3，Kling，Hailuo，Minimax等多个优秀模型 
- ✅ 4、文本卡片：支持Gemini3 Pro，GPT-5.2等多个优秀模型
- ✅ 5、表格卡片：支持数据收集和更新，支持数据分析
- ✅ 6、支持云端托管版 kardo.1mx.cn 和 本地安装版
- ✅ 7、支持多卡片链接，形成各种丰富创意内容，相比N8N更简单，比Coze更直接

#### 1.1.2026 - 预计2026.02.02 发布 (代码发布预计年后3月10号，很多兼容问题需要修复)
- ✅ 1、增加多个模型供应商选比，用户可以选择便宜及稳定的模型供应商，降低成本
- ✅ 2、增加海螺，字节的几个优秀模型支持
- ✅ 3、增加多图编辑和视频编辑的能力
- ✅ 4、增加专业镜头设置，分镜能力，脚本能力
- ✅ 5、增加声音卡片：支持音乐，人声，素材声生成
- ✅ 6、增加软件升级更新能力
- ✅ 7、增加工具能力，支持网络搜索，定向采集，视频拆片等自定义工具能力
- ✅ 8、增加数字人功能，带货等多个场景能力
- ⌛️ 9、开发自定义批量短剧，漫剧等默认工作流能力
- ✅ 10、增加优秀提示词模板，快速生成各种类型吸粉视频


#### 1.2.2603 - 预计2026.03.10 发布(代码预计4月10日发布)
- ⌛️ 1、保障稳定便宜的官方API模型渠道接入，支持非官方渠道的接入，用户按紧急度自由选择渠道，减少成本
- ⌛️ 2、开放已接入的稳定API，供有研发能力的用户自主调用，提供完整的计费和日志能力
- ⌛️ 3、一次性最多生成100个卡片的能力，包括图片，音频，视频等，抽卡概率大提升
- ⌛️ 4、新增Grok Image Video 成人级别内容生成（请遵守当地法律，慎重请勿传播）
- ⌛️ 5、新增资产库能力，包括常用图文、音视频，媒体账号等，支持自定义资产类别和属性，用户资产更丰富
- ⌛️ 6、支持硬件小智的接入，突破物理限制，AI超级流量入口
- ⌛️ 7、支持平台直接发单（短剧、广告片、短视频、营销图文、广告素材），创作者直接接单的能力，保障优秀创作者收益
- ⌛️ 8、支持自定义OEM，域名，品牌，Logo，文案，支付等信息全部自定义

### 技术栈
- 后端：Python FastAPI（事件流用 WebSocket）
- 流程引擎：LangGraph（Agent 循环 / 串并行 / 检查点）
- 本地数据库：SQLite + sqlite-vss（轻量 RAG）
- 前端：React + Vite + React Flow（画布 / DAG）

### 工程架构
- desktop
  - backend：桌面端后端（FastAPI）与应用启动器（pywebview）
    - app.py：REST/WS + 静态托管 + SQLite 持久化（workflows/nodes/edges）
    - launch_desktop.py：一键启动（dev/production），子线程运行后端
    - pyproject.toml：uv 项目配置（提供 desktop-backend/desktop-app/desktop-dev 脚本）
    - data/supra.db：本地 SQLite 数据库（运行自动创建）
  - frontend：桌面端前端（React + Vite + React Flow）
    - src/App.tsx：画布 + 运行事件 + 右侧属性面板
    - src/Canvas.tsx：三类节点、连线剪断/重连、工具条、尺寸调整
    - src/Inspector.tsx：节点属性编辑
    - vite.config.ts：开发代理与构建输出到 `../dist`
